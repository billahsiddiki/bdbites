Honest Assessment — Read This First

Kill Risk #1: Coordinated Inauthentic Behavior (CIB) Running "pro, anti, neutral" channels secretly under one entity will get you permanently banned. Meta's Q4 2025 transparency report removed 2,000+ CIB networks. If channels share the same IP, device fingerprints, ad accounts, or posting patterns — they get flagged and banned together. All of them. At once.

Instead: Run channels with distinct editorial voices — politics, lifestyle, entertainment, Bangla, English — all under one transparent parent brand. Different verticals, same parent. Legal, compliant, and stronger for trust.
Kill Risk #2: "Nobody will know it's AI" They will. And when they find out, credibility is destroyed overnight. The smarter play: don't market it as AI, don't hide it either. If asked, say "we use AI tools in our workflow like every modern newsroom." BBC, Reuters, AP all use AI now.
Kill Risk #3: "There's no copyright law in Bangladesh" Bangladesh is a Berne Convention signatory with the Copyright Act 2000. Platform-level enforcement (Content ID, DMCA) will take content down regardless. Instead: Aggregate and rewrite. Never copy-paste. Always credit sources. Use AI-generated visuals.
What Is Genuinely Strong About This Vision AI-first in a 20–25 person team market. No one in BD does full-pipeline automation. Bangla bite-size is wide open. 64M+ social users with median age 26. Neutrality as positioning in a hyper-polarized landscape. Self-improving system at zero additional cost.

Platform Algorithm Strategies

72M+ BD users — LARGEST AUDIENCE

  • 2026 priority: Meaningful interactions > comments > shares > reactions
  • Post type ranking: Video > Photo with long caption > Link posts
  • NEVER use link posts — post image, link in comments
  • Go Live during breaking news (massive priority boost)
  • Respond to EVERY comment in the first hour

~10M BD users — GROWTH PLATFORM (+46% YoY)

  • 2026: Reels > Carousels > Single images > Stories
  • Carousels: 1.4× more reach than single images
  • Max 3 hashtags (reduced hashtag weight, increased keyword weight)
  • Stories 8–10× per day with polls and quizzes

46.5M BD users — VIRAL ENGINE

  • 2026: Completion rate > shares > comments > likes
  • Sweet spot: 15–45 sec news, 30–90 entertainment
  • First 1 second MUST have visual movement
  • Text-on-screen critical — most BD users watch without sound
  • Never post identical content to TikTok and Instagram

44.6M BD users — MONETIZATION PLATFORM

  • Shorts threshold: 1K subs + 10M Shorts views in 90 days
  • Remove ALL watermarks from cross-posted content
  • Shorts algorithm favors new channels

Small but influential — journalists, politicians, diplomats

  • Thread format for explainers
  • Post 10–15 times/day (X rewards high frequency)
  • Where "pressure group" influence is felt among elite circles

Self-Improving Model

Every Monday at 6 AM BST, the system runs an automated review via n8n + Claude API: identifies top/bottom 10 posts, analyzes patterns, and outputs adjustment JSON applied directly to Supabase.

{ "scoring_weight_adjustments": { "emotion": +0.02, "timeliness": -0.01 }, "topic_boosts": { "cricket": +10, "celebrity_gossip": -5 }, "format_recommendations": { "carousel": "increase_to_40%" }, "posting_time_adjustments": { "instagram_evening": "20:45" }, "competitor_gaps": ["Topic X trending but no competitor covered"] }

Crisis Mode Protocol

Trigger: 3+ credible sources report the same major event within 15 minutes. Pause all scheduled content. Post updates every 5–10 min. All channels converge. NO speculation. NO graphic imagery. NO political blame. Post helpline numbers and donation links. Resume normal after 24–48h.

The 5 Kill Risks

#1: Bad AI Comment Goes Viral One tone-deaf reply gets screenshotted 50K times = brand finished. Phase 1: NO auto-replies. AI drafts → you review top 20 daily. Phase 2: Auto for low-risk only. Never auto-reply to political/religious topics.
#2: Publishing False Information One wrong fact on Day 5 = nobody trusts you on Day 50. Tier 1 sources: 5 min. Social media claims: NEVER as fact. Reverse image search for EVERY news image.
#3: Platform Account Suspension Use only official APIs. Never use Jarvee or unofficial automation. Remove ALL watermarks from cross-posted YouTube content.
#4: Bangla Writing Sounds Like AI Generate 50 samples. Show to 5 BD friends WITHOUT telling them it's AI. If more than 1 in 5 spots it, rewrite prompts. $0 test prevents $50K mistake.
#5: You Burn Out in Month 2 Minimum: ~45 min/day in Months 1–3, dropping to ~20 min/day by Month 6. Spot-check posts, review flagged comments, check alerts. This is not optional.
